# Global Population Analysis & Growth Trends (1970–2023)

## Overview
This project performs Exploratory Data Analysis (EDA) on global population data across 50+ years to identify growth patterns, density variations, and regional trends.

## Tools Used
- Python
- Pandas
- Matplotlib
- Seaborn

## Key Analysis
- Population growth trends (1970–2023)
- Continent-wise population distribution
- Population density comparison
- Bubble plot (Area vs Population)
- Boxplot (Density distribution)
- CAGR (Compound Annual Growth Rate)

## Key Insights
- Population is highly concentrated in a few countries
- Density varies significantly across continents
- Developing regions show higher population growth rates

## Dataset
Global population dataset including country-wise population, area, density, and growth rate.

1. Open the notebook in Jupyter
